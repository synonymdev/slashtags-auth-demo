# Slashtags Auth Demo	

Demonstrates signing in to websites with Slashtags

## usage	

- Run daemon

```	
npm i -g @synonymdev/slashtags-cli
slash -v
slash daemon start
```	

- Run backend server	

```	
cd backend/	
npm install	
npm start
```	

- Run the website's frontend	

```	
cd frontend/	
npm install	
npm start	
```	

Then visit `http://localhost:8001/`	

- Run the demo wallet	

```	
cd wallet/	
npm install	
npm start	
```	

Then visit `http://localhost:8080/`
